Wine Quality Prediction

Using Machine Learning to understand how physicochemical tests correlate with wine quality.

↓

The Objective

What I Did

🍷

The Problem

In the wine industry, predicting the quality of wine based solely on standard physicochemical tests (like pH, acidity, and alcohol content) can save significant time and money compared to human sensory panels.

🧠

The Approach

I built a supervised machine learning model using a dataset of red wine variants. By identifying nonlinear relationships between the chemical properties, the model can estimate the final quality score of a wine.

📈

The Outcome

A trained Random Forest algorithm that analyzes 11 different input parameters to successfully categorize the wine into quality classifications, highlighting the most important features driving wine quality.

Technology Stack

What I Used

🐼

Pandas & NumPy

Used for loading the dataset, handling missing values, and reshaping the data structure to prepare it for machine learning.

📊

Seaborn & Matplotlib

Utilized for Exploratory Data Analysis (EDA). I created pairplots, bar graphs, and correlation heatmaps to understand data distributions and feature relationships.

🤖

Scikit-Learn

The core machine learning library. I used `train_test_split` for data partitioning and `RandomForestClassifier` for the predictive model, along with accuracy metrics for evaluation.
